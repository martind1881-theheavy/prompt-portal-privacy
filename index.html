Privacy Policy
Prompt Portal

Chrome Extension by Innov8Edge  ·  Your AI Gateway

Last Updated: March 2026
Quick Summary

Prompt Portal stores everything locally on your device. Your prompts, categories, and GitHub credentials are saved only in your browser's encrypted local storage. No data is collected by Innov8Edge. When you use the GitHub backup feature, your prompt library is sent directly from your browser to GitHub using your personal access token — Innov8Edge never sees or touches it.

📋 1. Information We Collect

Prompt Portal collects and stores the following information locally on your device only, using Chrome's encrypted chrome.storage.local:

  • Prompt Library — titles, bodies, categories, and order of all prompts you create
  • Categories — the category names you define (Business, Education, Tech, and any custom ones)
  • GitHub Credentials — your personal access token, repository name, and file path for backup sync
  • Preferences — active category tab and other UI state

We do not collect your name, email address, browsing history, or any data beyond what is listed above.

⚙️ 2. How We Use Your Information

  • Prompt Library — displayed back to you within the extension for browsing, editing, and copying
  • Copy to Clipboard — when you click Copy Prompt or a Launch button, the prompt body is written to your clipboard
  • Launch Buttons — clicking a platform button opens that platform in a new tab. No prompt content is transmitted to Innov8Edge
  • GitHub Backup — when you push or auto-save, your prompt library is sent directly to GitHub's API using your token. Innov8Edge has no involvement in this transfer

💾 4. Data Storage

Local Storage Only

All data is stored on your device using chrome.storage.local — Chrome's encrypted, sandboxed storage. This data does not leave your device except when you actively use the GitHub backup feature.

GitHub Backup (Optional)

If you configure GitHub backup, your prompt library is pushed to a repository of your choosing using your personal access token. This transfer goes directly from your browser to GitHub — Innov8Edge has no server involved in this process and never receives a copy.

Auto-Push on Save

When GitHub credentials are configured, Prompt Portal automatically pushes your library to GitHub each time you save, edit, delete, or import prompts. This happens silently in the background. You can disable it at any time by clearing your GitHub credentials in Settings.

No Cloud Sync

Prompt Portal does not operate its own servers and does not back up your data to any Innov8Edge-controlled service. Your prompts exist only on your device and optionally in your own GitHub repository.

🚫 5. Data Sharing

We do not sell, rent, or share your data with any third parties.

Data leaves your device only in these cases:

  • GitHub API — when you push your prompt library via the backup feature (manual or auto-push)
  • Clipboard — prompt text is written to your local clipboard when you click Copy or a Launch button

No analytics platforms, advertising networks, or Innov8Edge servers receive any data from this extension.

✅ 6. Your Rights

View All prompts and categories are visible in the Library view

Export Download your full library as a JSON file via Settings
Delete Remove individual prompts or clear all data via Settings → Danger Zone
GitHub Credentials Clear your token and repo settings at any time in Settings → GitHub Backup
Uninstall Removing the extension from Chrome deletes all locally stored data automatically

🔒 7. Data Security

  • All GitHub communications use HTTPS encryption
  • GitHub credentials stored in Chrome's encrypted chrome.storage.local
  • Local-first architecture — your data stays on your device by default
  • No server-side processing, logging, or analytics by Innov8Edge
  • The extension is sandboxed by Chrome and cannot access other extensions or browser data
